枚举类型

枚举定义只能是定义整数类的值，其占用的字节数、有无符号性，通过基类标识指定。

如下所示：

enum Color:uint32{
    Red,
    Black,
}

结构体

结构体在airlang中是值类型，不会进入GC系统，除通过API分配独立的堆内存。

一般用于构成类中的共同属性。

struct Vec2{
    int32 x;
    int32 y;
}
struct Vec3 :Vec2{
    int32 z;
}

联合体

union Int32{
    int32 i32;
    struct{
        int8 [4] v4;
    }
}
